Installation
1. Unzip qaisarsatti-order_complete.zip to a local folder.
2. Copy unzipped folder 'qaisarsatti-order_complete' to your_wordpress_directory
/wp-content/ (If a different name then rename it to qaisarsatti-order_complete)
3. Click on plugin, you will see the plugin listed Qaisar Satti Autocomplete Order

User Guide
  Admin Panel
At the Admin Panel, go to plugin > Installed Plugin > Qaisar Satti Autocomplete Order. and activate the plugin
